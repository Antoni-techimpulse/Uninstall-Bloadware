# Security Policy

## Alcance

Este proyecto ejecuta cambios locales de administración en Windows 11. Cualquier reporte de seguridad debe incluir:

- versión del script;
- versión/build de Windows;
- pasos de reproducción;
- impacto observado;
- log de ejecución, eliminando datos personales.

## Recomendación

No ejecutar versiones modificadas del script sin revisión técnica previa.
