# Contributing

Las contribuciones deben seguir estos criterios:

- No eliminar componentes críticos de Windows por defecto.
- Mantener comportamiento interactivo salvo que se añada un modo parametrizado explícito.
- Documentar cada nueva app, servicio, tarea o clave de registro.
- Priorizar cambios idempotentes.
- Evitar dependencias externas.
